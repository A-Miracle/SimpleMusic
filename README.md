## 简介
可可音乐是一款开源Android本地音乐播放器，当前v 1.1.0，后续扩展在线播放

- 扫描播放本地音乐
- 歌词显示，自动搜索歌词
- 夜间模式，主题壁纸自定义
- **最新版:** [点击下载](https://raw.githubusercontent.com/A-Miracle/SimpleMusic/master/SimpleMusic-V1.1.0.apk)
